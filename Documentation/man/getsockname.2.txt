SSYYNNOOPPSSIISS

       iinntt ggeettssoocckknnaammee((iinntt _s_o_c_k_f_d,, ssttrruucctt ssoocckkaaddddrr **rreessttrriicctt _a_d_d_r,,

DDEESSCCRRIIPPTTIIOONN
       ggeettssoocckknnaammee() returns the current address to which the socket _s_o_c_k_f_d is bound, in the buffer pointed to
       by _a_d_d_r.  The _a_d_d_r_l_e_n argument should be initialized to indicate the amount of space (in bytes) pointed
       to by _a_d_d_r.  On return it contains the actual size of the socket address.

       The returned address is truncated if the buffer provided is too small; in this case, _a_d_d_r_l_e_n will
       return a value greater than was supplied to the call.

RREETTUURRNN VVAALLUUEE
       On success, zero is returned.  On error, -1 is returned, and _e_r_r_n_o is set to indicate the error.

EERRRROORRSS
       EEBBAADDFF  The argument _s_o_c_k_f_d is not a valid file descriptor.

       EEFFAAUULLTT The _a_d_d_r argument points to memory not in a valid part of the process address space.

       EEIINNVVAALL _a_d_d_r_l_e_n is invalid (e.g., is negative).

       EENNOOBBUUFFSS
              Insufficient resources were available in the system to perform the operation.

       EENNOOTTSSOOCCKK
              The file descriptor _s_o_c_k_f_d does not refer to a socket.
